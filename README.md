## ⚡ QA Performance Profiling & Resource Monitoring
이 저장소는 모바일 및 시스템 환경에서 앱 실행 중 발생하는 하드웨어 리소스(CPU, Memory, Battery temperature, Network) 데이터를 실시간으로 수집하고 분석하기 위한 프로파일링 스크립트를 포함하고 있습니다.

## 📂 Project Structure
- **android-resource-monitor**: Shell 스크립트 및 ADB 명령어를 활용한 안드로이드 기기 리소스(CPU 점유율, 메모리 릭 검사) 모니터링 도구

## 🛠 Tech Stack
- **Languages**: Shell Script (Bash)
- **Tools**: ADB
